# 카카오톡 단어 맞추기 솔버

카카오톡 단어 맞추기 게임을 위한 최적 전략 제시 도구입니다.  
3Blue1Brown의 정보이론 (엔트로피 최대화) 방식으로 매 시도마다 후보를 가장 많이 줄이는 단어를 추천합니다.

## 사용 방법

1. **자소 길이 선택** — 게임의 단어 길이(자소 수)를 선택합니다.
2. 솔버가 추천하는 단어를 카카오톡 단어 맞추기에 입력합니다.  
   추천 단어 대신 다른 단어를 사용했다면 입력창에 직접 입력할 수 있습니다.
3. 각 자모 타일을 탭하여 색상을 설정합니다.
   - **회색** — 해당 자모가 정답에 없음
   - **노랑** — 해당 자모가 있지만 위치가 다름
   - **초록** — 해당 자모가 정확한 위치에 있음
4. **결과 제출**을 누르면 다음 추천 단어를 확인하실 수 있습니다.
5. 추천 단어가 게임에 없는 단어라면 **밴** 버튼으로 제외할 수 있습니다.

## 기술 스택

- Next.js
- Tailwind CSS

## 로컬 실행

```shell
pnpm install
pnpm dev
```

`http://localhost:3000`에서 확인할 수 있습니다.

## 알고리즘

단어 목록은 [국립국어원 한국어 학습용 어휘 목록](https://www.korean.go.kr/front/etcData/etcDataView.do?mn_id=46&etc_seq=70&pageIndex=1) 에서 추출 및 가공한 3,786개 단어로 이루어져 있습니다.  
동사, 형용사, 관형사, 수사, 감탄사, 보조용언을 제외한 명사/대명사/부사/의존명사 위주로 필터링되었습니다.

지원하는 자소 길이는 2~16자소이며, 모든 단어는 자모 단위로 분해되어 저장됩니다 (예: `가게` → `ㄱㅏㄱㅓㅣ`).  
쌍자음(ㄲ→ㄱㄱ)과 겹모음(ㅐ→ㅏㅣ)은 기본 자소로 분해됩니다.

길이 선택 시 해당 길이의 전체 단어에 대해 엔트로피를 계산하여 최적의 첫 추천을 제공합니다.  
두 번째 시도부터는 현재 후보 단어들에 대해 엔트로피를 다시 계산하고, 가장 많은 정보를 제공하는 단어를 추천합니다.  
밴된 단어는 추천 풀에서 제외됩니다.
